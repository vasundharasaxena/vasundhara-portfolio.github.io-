Projects

Insight Generation with Flask

In this project, I developed a Flask web application for generating actionable insights from data. Users can upload datasets, apply analytical techniques, and visualize trends and patterns through an intuitive interface. The application offers descriptive statistics, correlation analysis, and predictive modeling, empowering users to make data-driven decisions.

Black Friday Sales Analysis

I performed exploratory data analysis (EDA) on Black Friday sales data to uncover customer purchasing patterns. By utilizing data visualization techniques, I analyzed product preferences, demographics, and the effects of discounts, providing actionable insights for retailers to optimize sales strategies during high-traffic shopping events.

Hotel Cancellation Prediction

I developed a machine learning model to predict hotel cancellations based on booking details. By analyzing factors like lead time and customer type, I identified patterns using EDA and feature engineering. The model, utilizing logistic regression and decision trees, optimizes resource management and minimizes revenue loss for hotels.

Pneumonia Detection using CNN

I developed a convolutional neural network (CNN) to detect pneumonia from chest X-ray images. By employing deep learning techniques like image preprocessing and data augmentation, the model achieved high accuracy in classifying pneumonia cases, assisting radiologists in early detection and treatment using TensorFlow and Keras.

London Bike Rides Visualization

I analyzed London’s bike-sharing data to uncover usage patterns related to ride duration, peak hours, and popular routes. Utilizing data visualization techniques, I examined the impact of factors like weather and station proximity, providing insights that can help city planners optimize bike-sharing services for better urban mobility.

Bot Detection Using Machine Learning

I developed a machine learning model to detect bot activity on online platforms by analyzing user behavior patterns. Using classification algorithms like Logistic Regression, Random Forest, and Gradient Boosting, I differentiated human from bot-generated activities. This solution enhances security and mitigates fraud on social media and e-commerce websites.

Driver Drowsiness Detection Using Deep Learning

I created a deep learning model to detect driver drowsiness in real-time by analyzing facial landmarks and eye movements. Utilizing a Convolutional Neural Network (CNN), the system triggers alerts for signs of fatigue, enhancing road safety. This project integrates OpenCV for video capture and TensorFlow for model implementation.

Tweets Analysis Using Sentiment Analysis and Visualization

I conducted a comprehensive analysis of tweets to gauge public sentiment on various topics. Using natural language processing (NLP), I extracted sentiment scores, categorized tweets, and visualized results with interactive graphs. This project provides valuable insights into public opinions and behaviors, applicable for brand monitoring and market research.
